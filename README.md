## рҹҺҜ Interview Prep

Each module has a dedicated `INTERVIEW_QUESTIONS.md` with questions
**targeted as we gain Java experience** вҖ” not definitions, but deep
internals, production tradeoffs, and design decisions.

**Collections topics:**
- HashMap internals вҖ” treeification, load factor, power-of-2 capacity
- ConcurrentHashMap vs synchronizedMap vs Hashtable
- hashCode + equals contract and what breaks silently when violated
- Fail-fast vs fail-safe вҖ” how `modCount` works under the hood
- LRU Cache design В· TimSort internals В· Thread safety tradeoffs

**Java 8 topics:**
- Lambda vs anonymous class at JVM level вҖ” invokedynamic, singleton non-capturing
- Stream lazy evaluation вҖ” vertical processing, short-circuit optimisation
- `thenApply` vs `thenCompose` вҖ” the flatMap parallel for CompletableFuture
- `orElse` vs `orElseGet` вҖ” why eager evaluation is a hidden performance trap
- Diamond problem resolution вҖ” `InterfaceName.super.method()` syntax
- `DateTimeFormatter` thread safety vs `SimpleDateFormat`
- Parallel stream pitfalls вҖ” commonPool starvation, shared mutable state

**Concurrency topics:**
- Race conditions, happens-before, synchronized vs volatile
- Deadlock вҖ” four conditions, consistent lock ordering, tryLock fix
- ReentrantLock vs synchronized, ReadWriteLock, StampedLock optimistic read
- Thread pool sizing вҖ” CPU-bound vs IO-bound
- ForkJoinPool work stealing, why not to block in common pool
- CAS, ABA problem, LongAdder vs AtomicLong under contention

**Java 9 topics:**
- `List.of` vs `Arrays.asList` вҖ” mutability, nulls, backing array
- `takeWhile` vs `filter` вҖ” order dependency, prefix semantics
- `var` вҖ” type inference rules, where it cannot be used
- `String` methods вҖ” `isBlank`, `strip`, `lines`, `repeat` (Java 11)
- `HttpClient` вҖ” async vs sync, Java 11 replacement for `HttpURLConnection`
- Records вҖ” immutability, canonical constructor, when to use vs regular class
- Sealed classes вҖ” exhaustive pattern matching, permitted subtypes
- Virtual threads вҖ” carrier threads, platform vs virtual, when to use
- Pattern matching `switch` вҖ” exhaustiveness, guards, type patterns

**Java 11 topics:**
- strip() vs trim() вҖ” Unicode whitespace handling
- lines() vs split("\\n") вҖ” line ending handling, Stream vs array
- HttpClient vs HttpURLConnection вҖ” async, HTTP/2, thread safety
- Predicate.not() вҖ” why it was needed, method reference negation
- var in lambda parameters вҖ” annotations, mixing rules

**Java 12-14 topics:**
- Switch statement vs switch expression вҖ” fall-through, exhaustiveness, yield
- yield vs return in switch blocks
- Text block indentation вҖ” how stripping works, closing """ position
- Pattern matching instanceof вҖ” scope rules, vs explicit cast
- Helpful NPEs вҖ” what changed, chained call debugging

**Java 15-16 topics:**
- Records вҖ” what compiler generates, compact constructor, vs immutable class
- Sealed classes вҖ” permits, final/sealed/non-sealed subtypes, exhaustive dispatch
- Records vs regular class вҖ” constraints, transparency, when to use each
- Stream.toList() vs collect(toList()) вҖ” mutability, null handling
- Sealed + pattern matching вҖ” algebraic data types, domain modeling
